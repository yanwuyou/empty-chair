# 空椅子 — 项目总结

> 创建时间：2026-05-25  
> 当前版本：v1.3 桌面分享版  
> 产品形态：单 HTML 文件，下载后双击即可使用

## 产品定位

**空椅子**是一个基于格式塔疗法“空椅子技术”设计的轻量自我表达工具。

使用者先选择一种接近当下状态的感受，把心里的话说出来；随后“换到对面的椅子”，试着从另一个位置回应刚才的自己。

核心承诺：

> 不追问，不评判，不催促。给自己一段不被打断的对话。

它不是聊天机器人，也不替代心理咨询、诊断或医疗服务。它只提供一个简单、私密的结构，帮助使用者完成一次“表达 → 换位 → 回应”。

## 使用方式

1. 下载 `空椅子.html`。
2. 双击文件，在电脑浏览器中打开。
3. 无需安装、登录或配置环境。
4. 对话记录只保存在当前浏览器的本地存储中，不上传。

移动 HTML 文件、更换浏览器或清理浏览器数据后，历史记录可能无法继续读取。重要记录可在历史页备份为 JSON，也可将单次对话导出为文本。

## v1.3 核心体验

### 桌面优先的两把椅子空间

- 左侧椅子用于表达当下感受。
- 右侧椅子用于回应刚才的自己。
- 通过位置、文案和视觉状态变化强化“换椅子”的仪式感。
- 窄窗口仍可正常使用，但主要体验为电脑浏览器设计。

### 两类感受入口

- 需要被听见：生气、担心、难过、紧张。
- 值得被记住：开心、勇敢、自豪、期待。
- 如果预设选项不合适，可以自定义一种当下感受，并继续完成相同的表达、换位与回应流程。
- 自定义情绪会正常保存到历史记录，并兼容文本导出、JSON 备份与恢复。

### 两种表达方式

- 跟着句子写：因为 / 我感觉 / 我希望。
- 自由写下来：不受模板限制。

两种模式切换时，已填写内容不会丢失。

### 理论与边界说明

首页可随时打开“了解空椅子”：

- 简要介绍格式塔疗法中的空椅子技术。
- 解释表达、换位、回应三步机制。
- 说明工具可以做什么、不能做什么。
- 说明本地隐私、记录保存与备份方式。

### 记录管理

- 查看历史对话。
- 单条记录导出为 TXT。
- 全部记录备份为 JSON。
- 从 JSON 恢复备份。
- 删除单条记录或清空全部记录。
- 三星记录本次换位是否带来启发。

## 安全与隐私边界

- 不要求登录。
- 不向网络上传对话内容。
- 所有用户输入在写入页面前进行转义，避免作为 HTML 执行。
- 记录只保存在当前浏览器本地。
- 明确提示：工具不替代心理咨询、诊断或医疗服务。
- 如果使用者处于强烈痛苦或存在伤害风险，引导其联系可信赖的人、当地急救或报警服务，以及全国统一心理援助热线 `12356`。

## 关键交互保护

- 空输入时给出明确反馈，不静默失效。
- 返回首页前，如存在未完成内容会先提醒。
- 删除与清空记录前二次确认。
- 切换表达方式保留草稿。
- 支持 `Esc` 关闭理论介绍或退出未完成对话。
- 支持系统“减少动态效果”设置。
- 页面文字可以正常选择、复制。

## 技术架构

- 单文件架构：HTML、CSS、JavaScript 全部内嵌。
- 零第三方依赖，无需服务器。
- 数据存储：浏览器 LocalStorage。
- 下载导出：浏览器 Blob API。
- 核心存储键保持为 `empty_chair_history`。
