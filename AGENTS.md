# AGENTS.md — 空椅子

## 项目概述

基于格式塔疗法“空椅子技术”的桌面优先单页自我表达工具。下载 HTML 后双击即可使用，无需安装、登录或联网。

## 产品文件

- `index.html` — 源文件，CSS/JS/HTML 全部内嵌
- `发布版/空椅子.html` — 对外分享文件
- `PROJECT_SUMMARY.md` — 产品定位、功能与发布说明
- `docs/` — 原始设计与实现计划

## 核心体验

1. 选择一种接近当下状态的感受
2. 在左侧椅子表达
3. 换到右侧椅子回应刚才的自己
4. 本地保存、回看或导出记录

## 修改注意事项

- 保持单 HTML、零依赖、双击即用。
- LocalStorage key 必须保持为 `empty_chair_history`。
- 工具不能宣称提供心理诊断、治疗或专业建议。
- 用户输入渲染前必须经过 `escapeHtml()`。
- 保持桌面优先，同时确保窄窗口无横向溢出。
- 不新增账号、后端、联网调用或 AI 对话，除非用户明确要求。
- 关键功能改动后必须完整验证：预设与自定义情绪、表达、切换模式、换椅子、保存、历史、导出、备份、恢复、删除与清空。
