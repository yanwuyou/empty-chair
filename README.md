# 空椅子 · 给自己一段不被打断的对话

> 基于格式塔疗法"空椅子技术"设计的轻量自我表达工具。不是一个聊天机器人，而是一把安静的空椅子。

一个不懂编程的人，用 AI 把这把椅子做出来了。单 HTML 文件，双击就能用。

## 这是什么

使用者先选择一种接近当下状态的感受，把心里的话说出来；随后"换到对面的椅子"，试着从另一个位置回应刚才的自己。

**不追问，不评判，不催促。** 对话内容存在本地浏览器里，不上传、不联网。

## 快速使用

1. 下载 `空椅子.html`
2. 双击打开（建议用 Chrome 或 Edge）
3. 选择一种感受，开始表达
4. 换椅子，回应自己

无需安装、注册或联网。

## 核心体验

- **两把椅子的桌面空间**：左椅子表达感受，右椅子回应自己
- **两类入口**：需要被听见（生气/担心/难过/紧张），值得被记住（开心/勇敢/自豪/期待）
- **两种表达方式**：跟着句子写，或自由写下来
- **记录管理**：历史查看、单条导出 TXT、全部备份 JSON、恢复备份
- **隐私安全**：不上传内容，不要求登录，记录仅存在当前浏览器

## 这不是什么

- 不是心理咨询，不替代诊断或医疗服务
- 不是聊天机器人，不会"替你想"
- 不会评判或催促你

## 技术

- 纯 HTML/CSS/JS，零外部依赖
- 数据存 LocalStorage，不上传
- 导出用 Blob API
