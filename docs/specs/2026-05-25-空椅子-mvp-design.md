# 空椅子 MVP — 设计文档

> 2026-05-25 | 版本 v1.0

## 产品概述

将心理学"空椅子练习"数字化为移动端 H5 应用。帮助存在情绪表达障碍的学生用户（80% 的人说不出自己的感觉）通过结构化引导，与情绪小怪兽对话并完成角色互换，从而理清内心感受。

**核心承诺：不追问、不评判、不催促。**

---

## 技术架构

| 项 | 选型 |
|------|------|
| 技术形态 | 单 HTML 文件，CSS/JS 内嵌 |
| 后端 | 无（MVP 不依赖 LLM） |
| 数据存储 | 浏览器 LocalStorage |
| 部署 | 静态托管（GitHub Pages / Vercel / 腾讯云 COS） |
| 入口 | 个人公众号菜单 → H5 链接 |
| 依赖 | 零外部依赖 |

---

## 视觉风格

**温暖手绘风** — 暖橙主调，圆润卡片，奶油白背景。

| 设计令牌 | 值 |
|------|------|
| 主色 | `#FF8C42`（暖橙） |
| 背景 | `#FFF5EC` / `#FFECD2` 渐变 |
| 卡片 | 白色 `#FFFFFF`，圆角 16-20px，柔和阴影 |
| 文字 | `#E65100`（标题）、`#BF8040`（辅助）、`#333`（正文） |
| 字体 | 系统默认中文字体，`font-weight: bold` 做标题层级 |
| 情绪色 | 生气 `#E65100`、担心 `#7B1FA2`、难过 `#1565C0`、紧张 `#2E7D32` |
| 动效 | 换椅子时小怪兽从左平移至右（CSS transition），约 300ms |

```
首页 ──→ 对话页 ──→ 完成页
(选怪兽)   (表达+换椅子)   (评分+记录)
```

## 页面 1：首页 — 小怪兽选择

### 默认首屏

- 顶部：空椅子 logo（🪑）+ 标题 + 副标题"选一只小怪兽，陪你说说话"
- 2×2 卡片网格，只展示 4 只消极情绪：

| 图标 | 名称 | 描述 | 情绪色 |
|------|------|------|--------|
| 😤 | 生气 | 心里有火 | 橙 |
| 😰 | 担心 | 心里不踏实 | 紫 |
| 😢 | 难过 | 想哭一会儿 | 蓝 |
| 😨 | 紧张 | 手心冒汗 | 绿 |

- 每张卡片：emoji 44px + 名称 + 一行情境描述
- 底部灰字："今天感觉还不错？看看开心小怪兽 👉"

### 展开积极情绪

- 点击底部文字，4 张消极卡片收起（或下滑展示），4 张积极卡片出现：

| 图标 | 名称 | 描述 |
|------|------|------|
| 😊 | 开心 | 嘴角上扬 |
| 🦁 | 勇敢 | 迈出一步 |
| 🌟 | 自豪 | 我做到了 |
| 🔭 | 期待 | 等不及了 |

- 底部文字变为"← 回不开心小怪兽"

## 页面 2：对话页 — 核心交互

### 步骤 1：小怪兽开场

- 选中的小怪兽居中展示（emoji 大图 + 名称）
- 下方气泡文案，按情绪类型变化：

| 情绪 | 开场白 |
|------|--------|
| 生气 | "嘿，我感觉到你很生气。跟我说说发生了什么吧？" |
| 担心 | "我在听。你心里有什么不踏实的，说出来给自己听听。" |
| 难过 | "没关系的，我在这里。你想到了什么？" |
| 紧张 | "深呼吸，慢慢说。你在紧张什么呢？" |

- 开心/自豪/期待/勇敢的开场白对应积极语境

### 步骤 2：用户表达

- 默认展示填空模板：
  ```
  因为 ___________________
  我感觉 _________________
  我希望 _________________
  ```
- 每个空行是一个 `<textarea>` 或 `contenteditable` 区域
- 底部两个切换按钮：`[📋 用模板填]`（当前）`[✍️ 自己写]`
- 点"自己写"→ 三个填空区合并为一个自由文本框，placeholder："想说什么都可以……"
- 点回"用模板"恢复填空格式
- 右下角"说好了"按钮提交

### 步骤 3：换椅子 + 回应

- 用户提交后，小怪兽 emoji 从左侧平滑移动到右侧（CSS animation ~300ms）
- 画面文字同时变化：
  - "现在，换你坐到椅子上。你想对自己说什么？"
- 下方模板：
  ```
  我想对自己说 ___________
  ```
- 同样支持切换"自己写"
- 提交后 → 跳转完成页

### 关键技术细节

- 换椅子动画：`transform: translateX()` 配合 `transition`
- 每步之间有 200-300ms 的淡入过渡
- 所有文字本地处理，不调任何 API

---

## 页面 3：完成页

- 动画展示本次对话的小怪兽 emoji
- 一句话总结："你和**生气小怪兽**聊了关于\*\*同桌扔你笔记本\*\*的事。"
  - 情绪名来自选中的小怪兽名称
  - 关键词取自模板填空第 1 行（"因为___"），直接拼接，不需要 NLP 提取
- 3 颗星评分（点星星交互，1-3 分）
- 两颗按钮：
  - `[再来一次]` — 回到首页
  - `[看看记录]` — 展示历史对话列表

## 预设引导语文案库

每只情绪小怪兽需要 3 组引导语：

1. **开场白**（步骤 1）：不同情绪不同文案
2. **倾听过渡语**（步骤 2 模板上方）：如"不着急，慢慢说。"
3. **换椅子引导**（步骤 3）：如"现在，换你坐到椅子上。如果你是这个生气的自己，你会对自己说什么？"

具体文案列表见附录 A（实现时整理）。

## 已知不做（MVP 范围外）

- LLM / AI 对话生成
- 语音输入
- 模块二（和未来的自己对话）
- 模块三（没说完的话）
- 用户系统 / 登录
- 云同步
- 微信支付

## 附录 A：完整引导语文案

### 消极情绪

| 情绪 | 开场白（步骤1） | 倾听过渡语（步骤2） | 换椅子引导（步骤3） |
|------|----------------|--------------------|--------------------|
| 😤 生气 | "嘿，我感觉到你很生气。跟我说说发生了什么吧？" | "不着急，把那股火说出来——因为什么？" | "现在换你坐到椅子上。如果你就是这股生气，你会对自己说什么？" |
| 😰 担心 | "我在听。你心里不踏实的东西，说出来给自己听听。" | "慢慢说，你担心的事，有名字了吗？" | "现在换你坐到椅子上。听了这些担心，你会怎么安慰自己？" |
| 😢 难过 | "没关系的，我在这里。你想到了什么？" | "让它流出来。你为什么会难过？" | "现在换你坐到椅子上。看着难过的自己，你想说什么？" |
| 😨 紧张 | "深呼吸。你在紧张什么呢？说出来会好一点。" | "慢慢说，手心出了汗也没关系。" | "现在换你坐到椅子上。冷静地看那个紧张的你，你会怎么安抚他？" |

### 积极情绪

| 情绪 | 开场白（步骤1） | 倾听过渡语（步骤2） | 换椅子引导（步骤3） |
|------|----------------|--------------------|--------------------|
| 😊 开心 | "笑容会传染！跟我说说，什么让你这么开心？" | "尽情说，快乐的细节都值得被记住。" | "现在换你坐到椅子上。记住这份开心——它来自哪里？你想提醒自己什么？" |
| 🦁 勇敢 | "你迈出了一步。跟我说说，你做了什么？" | "勇敢不等于不害怕。你越过的是什么？" | "现在换你坐到椅子上。看着勇敢的自己，你想给他什么力量？" |
| 🌟 自豪 | "你做到了！那份骄傲想让你说出什么？" | "不是炫耀，是告诉自己——我配得上这一刻。" | "现在换你坐到椅子上。这份自豪，会怎么鼓励未来的你？" |
| 🔭 期待 | "有期待真好。你等不及想去做什么？" | "想象它到来那一刻——你看见了什么？" | "现在换你坐到椅子上。面对这份期待，你提醒自己别忘了什么？" |

### 设计原则

- 不用任何 AI/外部调用，全部硬编码在前端
- 保持"引导"而非"分析"的语气——小怪兽不解读用户情绪，只帮用户说出来
- 每句话不超过 30 字，保持口语化
