# 我做了一把空椅子，给自己一段不被打断的对话

> 文章简介：一个由“空椅子理论”启发而来的单页 HTML 小工具。它不复杂，不需要安装，也不需要登录。只是给你一段安静的时间，把心里那些没有说完的话，先放到眼前，再换一个位置，重新看见自己。

![公众号封面](../05-视觉素材/文章配图/article-assets/2026-06-05-empty-chair-wechat/output/wechat-21x9-cover.png)

前几天，我看到一个视频。

视频里讲到，有一群孩子用“空椅子理论”做了一个微信小程序。

这个点一下子把我击中了。

因为“空椅子”这个概念，本身就很有画面感：你面前有一把椅子，椅子上可以坐着一个你想对话的人，也可以坐着过去的自己、某种情绪，或者一句一直没有说出口的话。

它不是让你立刻得到一个答案。

它更像是给自己留出一个位置：先把话说出来，再换到另一个角度，试着回应刚才的自己。

于是我就想，能不能做一个更简单的版本？

不做成复杂应用，不需要账号，不需要安装，不追求功能堆叠。就是一个 HTML 文件，点开就能用。像桌面上的一张纸，也像一把真的空椅子。

所以，我做了这个小工具，名字就叫：

**空椅子。**

## 为什么是一把空椅子？

![为什么是一把空椅子](../05-视觉素材/文章配图/article-assets/2026-06-05-empty-chair-wechat/output/article-01-theory.png)

“空椅子”源自格式塔疗法中的空椅子技术。

简单理解，它不是玄学，也不是说教，而是一种“表达、换位、回应”的练习结构。

有些话，我们平时很难完整说出来。

可能是因为没有合适的对象，可能是因为说出口会显得脆弱，也可能只是因为生活太快了，我们来不及认真听自己把话说完。

空椅子的价值，就在于它先创造了一个安全的表达位置。

你可以先站在自己的位置，把真实感受说出来。

然后，再“坐到对面的椅子上”，从另一个视角看刚才的自己，试着给出回应。

这个过程不一定解决问题，但它常常能让那些混在一起的情绪，变得更清楚一点。

## 我想把它做得很轻

![点开之后就是一间安静房间](../05-视觉素材/文章配图/article-assets/2026-06-05-empty-chair-wechat/output/article-02-app.png)

我做这个工具时，有一个很明确的想法：

**它应该简单到双击就能用。**

所以它不是小程序，也不是网页服务，更不是需要注册登录的产品。

它就是一个单页 HTML 文件。

下载到电脑上，双击打开，就可以开始。

整个界面也尽量克制：左边是一段说明，右边先让你选择此刻更接近自己的感受。

我没有把它做成“情绪测试”，因为我不想让用户一上来就被判断。

它只是问你：

**此刻，哪种感受更接近你？**

不必特别准确，只选一个最接近的开始就好。

## 真正重要的，是换一下位置

![换一下位置](../05-视觉素材/文章配图/article-assets/2026-06-05-empty-chair-wechat/output/article-03-flow.png)

工具里最核心的动作，是“换椅子”。

第一步，你先把心里想说的话写下来。

可以按提示写，也可以完全自由地写。

比如：

“因为这件事，我感觉很委屈。”

“我希望自己不要一直压着。”

“我其实只是想被认真听见一次。”

写完之后，工具会让你换到对面的椅子上。

这时候，你不再是刚才那个正在表达情绪的人，而是试着像回应一个在意的人那样，回应刚才的自己。

这一步很微妙。

很多时候，我们对别人能说出很温柔、很清楚的话，但对自己却很苛刻。

换一下位置，像是给自己开了一盏小灯。

不是为了讲大道理，而是为了让自己看见：原来我也可以被理解，我也可以被回应。

## 它很轻，但边界要清楚

![记录与边界](../05-视觉素材/文章配图/article-assets/2026-06-05-empty-chair-wechat/output/article-04-record.png)

我也给这个小工具加了一些必要的边界。

它不是心理咨询，也不是诊断或治疗服务。

它更适合作为一个自我表达与整理工具：在情绪没那么汹涌、你只是想安静梳理一下自己的时候，用它把话说完整。

记录只保存在当前浏览器里，不会上传。

你可以删除单条记录，也可以清空全部记录；如果想留下来，也可以导出文本，或者备份恢复。

我希望它是一个轻工具，但不是一个随便的工具。

简单，不代表粗糙。

## 为什么我想把它分享出来？

因为我越来越觉得，很多工具真正有价值的地方，不一定在于它有多复杂。

有时候，一个小工具只要能帮人停下来几分钟，就已经有意义了。

我们每天都在接收很多信息，也在处理很多关系。

但留给自己说话的时间，反而越来越少。

这把空椅子，不会替你做决定，也不会替你解决所有问题。

它只是安静地摆在那里。

当你想把一些话说出来的时候，它给你一个入口。

当你想换个角度看看自己时，它给你一个位置。

当你想把这段对话留住时，它给你一份记录。
